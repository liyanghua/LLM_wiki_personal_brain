# 6大维度·42个细分变量选择逻辑01

# 类目专属·穷尽测图变量清单（6大维度·42个细分变量）

## 维度1：视觉核心层（决定第一眼停留）

**选择逻辑**：a：匹配店铺视觉体系。b：匹配人群（例如男孩和女孩做不同风格）。

**通用匹配原则与方法**：所有变量选择需先遵循“店铺视觉体系优先”，确保测图风格与店铺首页、详情页、其他产品主图的视觉统一（如店铺主打森系简约，优先选择纯色背景、低饱和色调、叶子相关元素）；再结合人群特征做细分适配，针对男孩/女孩、低龄/高龄儿童的审美差异，调整变量组合，实现“视觉统一+人群精准”，避免测图风格杂乱，同时提升目标人群的第一眼好感度。

| 变量类别 | 细分变量 | 可选组合（参考选择逻辑） | 适配场景 | 匹配原则与方法（结合选择逻辑a+b） |

| 背景选择 | 纯色背景 | 奶白/浅灰/莫兰迪蓝/马卡龙粉（儿童友好色） | 突出产品质感，适配护眼卖点 | a. 若店铺视觉主打简约、高级、护眼，优先选奶白/浅灰；主打活泼、童趣，优先选莫兰迪蓝/马卡龙粉，保持与店铺首页底色、详情页背景统一。b. 男孩适配莫兰迪蓝，女孩适配马卡龙粉，低龄儿童可选择饱和度稍高的浅色系，高龄儿童适配低饱和的奶白/浅灰。 |
|  | 场景化背景 | 儿童书桌实景（搭配台灯/书本/文具） | 还原使用场景，增强代入感 | a. 匹配店铺场景化视觉风格（如店铺详情页多为居家学习场景图，主图场景需保持一致，台灯、文具款式与店铺详情页展示款统一）。b. 低龄儿童场景搭配卡通文具、色彩明亮的台灯；高龄儿童场景搭配简约文具、护眼台灯，贴合不同年龄段学习场景习惯。 |
|  | 卡通背景 | 森林/海洋/太空/恐龙/公主主题（贴合孩子喜好） | 主打颜值卖点，吸引儿童注意力 | a. 若店铺视觉有固定卡通风格（如主打森系，优先选森林主题；主打童趣，可任选适配风格），避免与店铺整体视觉冲突。b. 男孩优先选森林/海洋/太空/恐龙主题，女孩优先选森林/公主主题；低龄儿童选图案圆润、色彩鲜艳的卡通背景，高龄儿童选图案简约、色调柔和的卡通背景。 |
| 产品视角 | 整体全景 | 桌垫平铺全貌（含尺寸标注） | 展示产品完整形态、尺寸适配性 | a. 若店铺所有产品主图均以全景为主（统一视觉规范），优先选择此视角；若店铺主打细节展示，可将全景作为辅助视角。b. 针对注重尺寸适配的家长（多为高龄儿童家长，关注桌垫与书桌匹配度），全景需清晰标注尺寸；低龄儿童家长更关注颜值，全景可搭配简约装饰，弱化尺寸标注。 |
|  | 局部特写 | 桌垫边缘（防卷边）、材质纹理（磨砂/硅胶质感）、防水测试（水滴滚落） | 突出材质、工艺、功能细节 | a. 匹配店铺“重品质、重细节”的视觉定位（如店铺详情页多细节特写图，主图特写风格需统一，修图质感、光线与详情页一致）。b. 注重材质安全、耐用性的家长（全年龄段通用），优先特写材质纹理、防卷边细节；有防水需求的家长（低龄儿童，易弄脏桌垫），优先特写防水测试场景。 |
|  | 动态视角 | 桌垫卷曲回弹、孩子书写/画画动态抓拍 | 展示耐造、柔软特性 | a. 若店铺视觉主打“真实使用场景”，优先选择此视角，动态画面风格与店铺详情页视频、场景图保持一致（如光线、色调统一）。b. 低龄儿童适配孩子画画动态，突出童趣；高龄儿童适配书写动态，突出实用；针对注重“耐造”的家长（多为有低龄儿童、多孩家庭），优先选择卷曲回弹动态。 |
| 构图方式 | 满版构图 | 桌垫占画面80%以上，无多余元素 | 突出产品本身，适配纯色背景 | a. 匹配店铺“简约、聚焦产品”的视觉体系（如店铺主图多为满版构图，无多余装饰），保持构图风格统一。b. 针对注重产品本身（材质、图案）的家长，优先选择；低龄儿童可适当减少满版占比，增加轻微装饰元素，避免单调。 |
|  | 三分构图 | 桌垫居左/右，右侧留空（可加文案）,桌垫居左/右，左侧留空（可加文案）；同理：上下构图、斜对角构图等。（核心是 \*\*“黄金分割” 简化版 \*\*，将画面用 2 条水平、2 条垂直辅助线，平均分割成 9 个均等区域（3×3 网格），核心主体（桌垫）放置在辅助线交点或辅助线上，而非画面正中心。<br>） | 搭配卖点文案，信息清晰 | a. 若店铺主图多带卖点文案，优先选择此构图，文案位置、字体、颜色与店铺其他主图保持一致（如统一放在右侧留空处，字体为简约黑体）。b. 高龄儿童家长更关注卖点信息，适合此构图；低龄儿童家长更关注视觉，可减少文案篇幅，留空处搭配简约叶子装饰。 |
|  | 场景嵌套构图 | 孩子使用桌垫的完整场景（含人物） | 主打场景化卖点 | a. 匹配店铺“场景化营销”视觉体系（如店铺首页、详情页多有人物场景图），人物穿搭、场景布置与店铺风格统一（如森系场景搭配棉麻服饰、绿植）。b. 低龄儿童适配单人可爱场景，高龄儿童适配简约学习场景，多孩家庭可适配双人场景，贴合不同人群的使用场景需求。 |
| 光影色调 | 明亮通透光 | 自然光/柔光箱，画面干净明亮 | 适配护眼、安全卖点 | a. 若店铺视觉主打“清新、护眼”，优先选择此色调，光线亮度、通透度与店铺详情页产品图保持一致，避免光线过暗或过亮。b. 全年龄段人群通用，尤其适配注重护眼的家长；男孩、女孩均可使用，无需刻意区分，重点保证画面干净。 |
|  | 柔和暖调光 | 暖黄调，营造温馨学习氛围 | 适配居家场景、亲子卖点 | a. 匹配店铺“温馨、居家”的视觉体系（如店铺主图多为暖色调，营造亲子氛围），暖调色温与店铺其他产品图统一（避免色差过大）。b. 低龄儿童、亲子场景优先选择，贴合宝妈对“温馨学习环境”的诉求；女孩场景可适当提升暖调饱和度，男孩场景保持柔和低饱和暖调。 |
| 画质质感 | 高清锐度 | 纹理清晰可见（硅胶颗粒/印花细节） | 突出材质高级感 | a. 匹配店铺“高品质、高端”的视觉定位（如店铺产品主打高端材质，主图画质需高清锐度，突出材质细节），修图风格与店铺详情页细节图统一，避免过度磨皮。b. 针对注重材质高级感的家长（多为高龄儿童家长、高端客群），优先选择；叶子图案款需突出叶片纹理，提升质感。 |
|  | 真实质感 | 轻微使用痕迹（展示耐脏特性） | 主打耐磨、耐造卖点 | a. 若店铺视觉主打“真实、实用”，优先选择此质感，轻微使用痕迹的呈现方式（如轻微污渍、轻微折痕）与店铺详情页“使用场景图”保持一致，避免过于杂乱。b. 针对注重耐造、耐脏的家长（多为低龄儿童家长、多孩家庭），优先选择；男孩场景可适当增加轻微使用痕迹，更贴合真实使用场景。 |
